편의점 할인정보 사이트

편의점의 할인정보를 제공해주는 어플리케이션
편의점별로 분류된 다른 APP과 달리 제품별로 카테고리를 분류시켜
원하는 제품을 보다 쉽고 빠르게 찾을 수 있도록 하였다.

ROLE

Planning

Design

Publishing

ISSUE

편의점마다 카테고리를 나누어
할인 정보를 제공하는
기존 편의점 할인정보 어플

PROBLEM

제품명으로 검색할 수 없고
카테고리별로만 나누어져 있어서
원하는 상품의 할인 정보를 얻기 불편

SOLUTION

제품 카테고리별로 나누어 할인 정보를
제공함으로써 보다 편리하게 할인정보를
얻을 수 있도록 함

CONCEPT

더 쉽게 비교해서 더 싸게 사자

LOGO

더 쉽게 비교하고 더 싸게 사자는 컨셉으로
the(더)편의점으로 네이밍을 정하였고
the의 t에 더한다는 의미의 +기호를 시각적으로 표현

COLOR

특정 편의점이 연상되지 않도록 타 편의점과 중복되지 않은 색을 사용하면서
수많은 할인정보들 안에서 원하는 정보를 찾을 수 있다는 점에서
주목성과 강조의 성격이 강한 노랑색을 main color로 사용
